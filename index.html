Overview

When I found the original source from codepen and it was using a hard coded Array data and filter directive for filtering the data. Then I decided to make it little challenging and to learn something from it. First step I made was read the data from attached JSON data file. It worked well. Still didn't satisfied what have done. I needed something more interesting that I don't need to write a hard coded data. So I decided to attach a external JSON data from someones server. In the process I have learnt about JSON type difference on Angularjs. started with 10 lines of code then it shrinked into one line code. It was pretty awesome practice.

Tech specs:

Twitter bootstrap, Angularjs, HTML5, CSS3

Additional:

ng- directives and ng- basics like app module, controller, templating etc

Solution:

I've done two step improvement done on this example. First I've converted the hard coded array data into json file and then implement into the code by following method:
$http.get({'file.json'})
.success(function(data) {...}
After this step I needed to pull data from different server. Therefore needed a little change one this line
$http.jsonp('URL.json?callback=JSON_CALLBACK' )
.then( function ( response ) {..}
